# Dietalife

Dietalife is a Persian health and nutrition web application focused on diet planning and BMI analysis.

The project was built as a lightweight responsive frontend application using pure HTML, CSS and JavaScript without external frameworks.

## Features

- BMI calculator
- Smart diet recommendation system
- Medical and nutrition related pages
