# Dietalife

Dietalife is a Persian health and nutrition web application focused on diet planning and BMI analysis.

The project was built as a lightweight responsive frontend application using pure HTML, CSS and JavaScript without external frameworks.

## Features

- BMI calculator
- Smart diet recommendation system
- Responsive design for mobile and desktop
- Persian RTL interface
- Medical and nutrition related pages
- Simple and modern UI

## Technologies

- HTML
- CSS
- JavaScript

## Project Structure

```bash
Css/
Js/

index.html

```

## Run Locally

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/dietalife.git
```

Open `index.html` in your browser.

## Notes

This project is currently frontend-only and does not require a backend or database.

The diet system and BMI calculator are implemented using vanilla JavaScript logic.
